# **App Name**: SmartSales AI

## Core Features:

- Client Management: CRUD operations for clients with details like company name, contact info, address (geocoded), and consumption data.
- Proposal Generation: Create, edit, and manage sales proposals with client-specific information.
- AI-Powered Proposal Drafts: Generate initial proposal drafts using Genkit AI based on client details and recommended plans. Genkit will act as a tool for suggesting proposal copy.
- Commission Tracking: Track and manage sales commissions based on accepted proposals or confirmed orders, with calculations performed in Cloud Functions.
- Sales Material Repository: Centralized repository for sales materials (PDFs, images, links) accessible to sales users.
- Admin Dashboard: Admin interface to view and manage all sales teams, clients, proposals, and commissions across the organization.
- User Authentication and Roles: Firebase Authentication to handle user sign-up, login, and role-based access control (sales, admin) using custom claims.

## Style Guidelines:

- Primary color: Teal (#20c997) evoking growth and reliability, nodding to the 'Refill' aspect of the product.
- Background color: Very light green (#F1F8E9), a subtle backdrop ensuring readability and visual comfort.
- Accent color: Golden yellow (#FDD835) draws attention to key actions and elements, signifying value and opportunity.
- Font family: 'Manrope' for headlines and body text; a modern, geometric sans-serif font that maintains readability and reflects the brand's forward-thinking approach.
- Consistent set of icons for navigation and actions, using a minimalist style for clarity.
- Clean and intuitive layout with clear visual hierarchy, using side navigation and a header with quick actions.
- Subtle transitions and animations to enhance user experience without being distracting.
- Dashboard layout with cards displaying key metrics (revenue, expenses, sales) and a list of recent messages (similar to image 1).
- Proposal creation interface with clear sections for client info, plan selection, and AI-generated draft, aiming for a user-friendly and efficient workflow (inspired by image 2).
